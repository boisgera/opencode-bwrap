# opencode-sandboxed

A sandboxed environment for running opencode safely.

## Installation

```bash
./install.sh
```

This will install the `opencode-sandboxed` binary to `~/.local/bin` (or `$XDG_DATA_HOME/bin` if set).

## Usage

Run the sandboxed opencode binary directly:

```bash
opencode-sandboxed
```

## Requirements

- [opencode](https://github.com/anomalyco/opencode) installed on the system
- Bash
- `~/.local/bin` (or `$XDG_DATA_HOME/bin`) in your PATH
